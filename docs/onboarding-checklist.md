# NexaCore Technologies -- New Employee IT Onboarding Checklist

**Document Version:** 1.0  
**Last Updated:** May 2026  
**Owner:** IT Administrator (Nishant)  
**Reference Ticket:** #462201

---

## Purpose

This checklist is used by the IT team to ensure every new employee at NexaCore Technologies receives a complete and consistent IT setup before their first day. It is raised as a SEV-1 ticket and assigned directly to the Systems Administration team.

---

## Pre-Arrival (Complete Before Start Date)

### User Account Setup
- [ ] Create Active Directory / system user account
- [ ] Set temporary password following password policy
- [ ] Force password reset on first login
- [ ] Add user to correct department group
- [ ] Assign role-based access permissions

### Email Configuration
- [ ] Create company email address (firstname.lastname@nexacore.in)
- [ ] Configure email client on workstation
- [ ] Test send and receive functionality
- [ ] Add user to relevant department mailing lists

### System Access and Permissions
- [ ] Grant access to shared network drives relevant to department
- [ ] Set up folder permissions (read / write as appropriate)
- [ ] Configure access to internal systems and tools
- [ ] Create helpdesk user account in osTicket

---

## Hardware Setup

### Workstation
- [ ] Assign laptop or desktop from IT inventory
- [ ] Log asset tag and serial number against employee record
- [ ] Install and update operating system
- [ ] Run Windows Update and install all patches
- [ ] Configure display settings and dual monitor if required
- [ ] Test keyboard, mouse, and all peripherals

### Networking
- [ ] Connect workstation to company network
- [ ] Confirm network drive auto-mapping is working
- [ ] Test internet connectivity

---

## Software Installation

- [ ] Microsoft Office / 365 suite
- [ ] VPN client (configured with company credentials)
- [ ] Company-approved browser with bookmarks
- [ ] Any department-specific software (confirm with manager)
- [ ] Antivirus / endpoint protection
- [ ] PDF reader

---

## VPN and Remote Access

- [ ] Install and configure VPN client
- [ ] Create VPN credentials for the user
- [ ] Test VPN connection from outside the office network
- [ ] Provide user with VPN login instructions document

---

## Security Briefing

- [ ] Walk user through password policy (minimum 12 characters, change every 90 days)
- [ ] Explain acceptable use policy for company devices
- [ ] Brief user on phishing awareness and how to report suspicious emails
- [ ] Confirm user understands not to install unauthorised software
- [ ] Provide IT helpdesk portal URL and explain how to raise a ticket

---

## Handover and Sign-Off

- [ ] Provide user with welcome pack including login credentials
- [ ] Confirm user can log in successfully
- [ ] Confirm email is working
- [ ] Confirm VPN is working
- [ ] Confirm network drive access is working
- [ ] Walk user through how to submit a helpdesk ticket
