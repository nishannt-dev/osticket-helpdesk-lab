## Internal Notes (Agent: Maria Lopez)

Received ticket from Maria Hill reporting Outlook not displaying emails. Steps taken:

1. Confirmed internet connectivity on user's machine -- browsing was working fine, ruled out network issue
2. Checked that the mail server was reachable -- no server-side outages reported
3. Attempted to manually send/receive in Outlook -- returned an error referencing the profile
4. Navigated to `%localappdata%\Microsoft\Outlook` and cleared the OST cache file
5. Restarted Outlook -- issue persisted, emails still not loading
6. Ran Outlook profile diagnostic via Control Panel > Mail > Email Accounts
7. Found the existing profile was flagged as corrupted
8. Created a new Outlook profile and configured it with Maria's email account
9. Allowed the new profile to sync -- all emails loaded successfully within 5 minutes
10. Confirmed old profile was removed to prevent confusion

Root cause: The Outlook profile became corrupted, likely due to an improper shutdown the previous day.
