## Issue Reported

> "My computer has been extremely slow since the last Windows update. Everything takes forever to open."

---

## Internal Notes (Agent: James Carter)

Received ticket from Tom Baker reporting significant performance degradation following a Windows update. Steps taken:

1. Remotely accessed Tom's workstation to assess performance
2. Opened Task Manager -- CPU was at 78%, RAM at 94% on startup
3. Checked startup programs -- found 14 applications set to run on startup including several that were installed during the recent Windows update
4. Ran Disk Cleanup to clear temporary files -- freed up 4.2 GB of disk space
5. Ran a malware scan using Windows Defender -- no threats found, ruled out malware as a cause
6. Disabled 9 unnecessary startup applications via Task Manager > Startup tab
7. Cleared Windows Update cache as post-update cleanup
8. Restarted the machine and monitored performance -- CPU dropped to 12% at idle, RAM to 41%
9. Opened several applications to verify response times -- all back to normal
10. Confirmed with Tom that the machine felt noticeably faster

Root cause: A Windows update enabled several background services and added new startup programs without user awareness, causing high CPU and RAM usage on boot.

---

## Resolution Reply (Agent: James Carter)

> Hi Tom, the slowness was caused by the recent Windows update enabling a number of background services and startup programs that were not needed. I have cleared these out, run a disk cleanup, and optimised your startup process. Your computer should feel back to normal now. Please let us know if you notice any further slowness.
